# Re:Sputnik — бриф для презентации

> Документ для дизайн-команды (Claude Design): идея, позиционирование и список
> функций, на основе которых нужно собрать презентацию продукта.
> Язык интерфейса и аудитории — русский.

---

## 1. Что это в одном предложении

**Re:Sputnik** — настольное приложение (Windows и macOS), которое настраивает и
обслуживает прокси-роутер на OpenWRT **без терминала и без знания командной
строки**: пользователь проходит пошаговый мастер, а программа всё делает за него
по SSH.

Подзаголовок для обложки:
> «Спутник вашего роутера: настройка обхода блокировок в несколько кликов».

Название обыгрывает двойной смысл: **спутник** = и «компаньон» (companion), и
«орбитальный спутник». Семейный префикс **Re:** связывает продукт с
**Re:HomeProxy** (роутерная часть).

---

## 2. Проблема, которую решаем

Сегодня, чтобы поднять надёжный обход блокировок на домашнем роутере, нужно:

- зайти по SSH, править конфиги, ставить пакеты вручную;
- разбираться в sing-box / Hiddify / ByeDPI / zapret, ядрах, протоколах;
- вручную импортировать подписки, прописывать маршруты и списки;
- чинить то, что отвалилось, читая логи.

Это барьер для **99% обычных пользователей**. Технические гайды не масштабируются.

**Re:Sputnik убирает терминал целиком.** Пользователь видит человеческий мастер,
кнопки и понятные подсказки. Вся «магия» (SSH, ubus, uci, установка ядер,
диагностика) скрыта под капотом и происходит автоматически.

---

## 3. Для кого

- **Основная аудитория — неподготовленные пользователи.** Купил роутер, хочет,
  чтобы «просто работало». Не знает, что такое SSH.
- Энтузиасты, которым надоело руками поднимать конфиги на каждом устройстве.
- Три пути входа в экосистему (это важно показать на слайде «как начать»):
  1. **Десктоп-приложение** (Re:Sputnik) — для большинства;
  2. **AI-агент настройки** — диалоговый помощник;
  3. **SSH + LuCI** — для технических пользователей вручную.

---

## 4. Ключевая идея архитектуры (для слайда «почему это надёжно»)

- **Никакого ИИ в рантайме настройки.** Всё, о чём раньше «думал» ассистент,
  превращено либо в детерминированное правило движка, либо в явный выбор в
  интерфейсе. Поведение предсказуемо и воспроизводимо.
- **Движок-first, UI поверх.** Одна и та же логика обслуживает и быстрый мастер,
  и расширенный режим.
- **`RouterClient` — единственная дверь к роутеру.** Все действия идут через один
  контролируемый SSH-канал (paramiko).
- **SecurityGate — один «шлюз согласия» над каждым действием.** Прозрачность:
  что делается, откуда берётся, что пишется в лог.
- **Проверки связи — это «глаза» приложения.** `connection_check`,
  `byedpi_strategy_test` дают машиночитаемые сигналы вместо человеческой оценки
  «работает / не работает».
- Секреты роутера хранятся в **системном хранилище ключей ОС** (Keychain /
  Credential Manager), а не в файлах.
- По сути это **универсальная платформа «поставь и настрой ПО на OpenWRT»**;
  Re:HomeProxy — лишь первый «рецепт».

## 5. Два режима работы (главный слайд продукта)

### ⚡ Быстрая настройка (Quick Setup)
Линейный мастер для новичка. Ведёт за руку:
**Подключение → Первичная настройка → Интернет → Установка ПО →
Импорт серверов/подписки → Wi-Fi → Проверка → Завершение**.
Сам подставляет разумные значения по умолчанию.

### ⚙ Расширенный режим (Advanced)
Свободная навигация по разделам для тех, кто хочет контроля:
Обзор, Серверы и подписки, Правила, Контроль доступа, ByeDPI, Ядро,
Диагностика, Дополнительно.

## 6. Список функций по экранам (карта для слайдов и скриншотов)

### Мастер (Quick Setup)
| Экран | Что делает |
|---|---|
| **Подключение** | Находит роутер в сети, подключается по SSH; список «сохранённых роутеров» (как в WinBox), флажок «Запомнить роутер». |
| **Первичная настройка** | Задаёт надёжный (максимально случайный) пароль администратора; базовая безопасность. |
| **Интернет** | Проверяет/настраивает выход в интернет (кабель/Wi-Fi-мост); кнопка «Назад» на шаг подключения. |
| **Установка ПО** | Ставит ядра и пакеты с официальных источников (opkg/apk), сама подбирает релиз под версию OpenWRT и архитектуру. |
| **Серверы / Подписки** | Импорт подписки, share-ссылок, `.conf` (WG/AWG), `vpn://` (Amnezia); парсинг VLESS/Hysteria2/Trojan/SS и пр. |
| **Wi-Fi (точка доступа)** | Настройка SSID/пароля/шифрования; QR-код для подключения телефона. |
| **Проверка** | Реальные тесты связи через прокси (многохостовая проверка YouTube/Telegram/Discord/Speedtest). |
| **Завершение / Безопасность** | Итоги, защита доступа, аренда SSH-ключа (renewable). |

### Расширенный режим
| Раздел | Что делает |
|---|---|
| **Обзор** | Живой дашборд: модель/аптайм/IP, CPU/ОЗУ, WAN, активный сервер и пинг, ядро, статус ByeDPI, пул серверов, QR Wi-Fi. |
| **Серверы и подписки** | Управление узлами, обновление подписок, URLTest-пул «лучший сервер». |
| **Правила** | Маршрутизация: списки Re:Filter / Russia Inside, сервисные списки; режим по умолчанию. |
| **Контроль доступа** | 1 глобальный режим фильтрации + 3 независимых списка на устройство (фильтр / игровой TCP / весь трафик). |
| **ByeDPI** | Управление обходом DPI, генератор/тестер стратегий, переключатель «Блокировать QUIC» (добавляет правило nft на WAN). |
| **Ядро** | Выбор и переключение ядра (sing-box-extended / Hiddify), их совместимость с протоколами. |
| **Диагностика** | Проверки связи, статусы, Clash API, логи — «приборная панель». |
| **Дополнительно** | Имя роутера, **резервная копия/восстановление**, сброс к заводским, LAN/DHCP и постоянные IP устройств, **редактор Wi-Fi по каждому радио** (SSID/пароль/шифрование/канал/ширина с реальным поллингом поддерживаемых каналов), **UPnP** и **SQM** (с кнопкой «Измерить скорость»). |

## 7. Фишки, которые стоит подсветить отдельно (слайды «вау»)

- **QR-код Wi-Fi** прямо в приложении — навёл телефон, подключился.
- **Резервная копия и восстановление роутера** в один клик (как в WinBox).
- **Постоянный IP для устройства** (статическая аренда DHCP) из удобного списка.
- **Редактор Wi-Fi с реальными каналами** — приложение опрашивает каждое радио
  (включая 6 ГГц / Wi-Fi 7) и предлагает только реально поддерживаемые каналы и
  ширину, как это делает LuCI. Статус каждого радио — зелёный/красный/оранжевый.
- **Кнопка «Измерить скорость»** для SQM — реальный замер на RU-дружественных
  хостах, авто-подстановка значений.
- **Генератор и тестер стратегий ByeDPI** — проверяет обход DPI на нескольких
  площадках и показывает результат точками ●●●○.
- **Сохранённые роутеры** — мульти-роутерный workflow, секреты в ОС-хранилище.

## 8. Тон и визуальный язык (для дизайнера)

- **Современно, спокойно, доверительно.** Это инструмент про безопасность и
  «просто работает», не про хакерство и зелёный текст на чёрном.
- Тёмная тема — основная (приложение по умолчанию в dark).
- Интерфейс собран на **customtkinter** (Tk): крупные карточки, скруглённые
  элементы, статус-точки (зелёный/красный/оранжевый), мягкие акценты.
- Логотип/брендинг лежат в `src/re_sputnik/resources/branding` (SVG + PNG/ICO/ICNS).
- Иконки: Lucide/Phosphor для UI, Simple Icons для брендов сервисов.
- **Дисклеймер о товарных знаках обязателен** на финальном слайде: продукт не
  аффилирован с YouTube/Telegram/Discord/Meta/Hiddify; логотипы сервисов — лишь
  для идентификации.

## 9. Дистрибуция (слайд «как получить»)

- Один портативный файл: **`.exe` (Windows)** и **`.app` (macOS, Apple Silicon)** —
  без установщика, перетащил и запустил.
